HTML이란 무엇인가?

태그를 알면 인생이 달라진다.

인생은 태그를 알기 전과 태그를 알기 후로 나뉜다. 솔직히 아직 그정도 의미인지 까진 모르겠고 쉽게 생각해서 의식주 같은 느낌인거 같다. 없으면 살아갈 수 없는, 없으면 유의미한 코딩을 하기 어려운 가장 기초이면서 젠가의 첫번째 다리를 맡고 있는 녀석.

그래서 지금 지금까지 배운 태그를 정리하는 시간을 가지고 있다.
  • 1. Strong = 글자를 강조해준다.
  • 2. u = 밑줄 칠수있음
  • 3. h1~h6 = 제목을 만든다.
  • 4. br = 줄바꿈 한칸
  • 5. p = 단락 구분(단락 앞뒤로 한칸씩 줄바꿈)
  • 6. li = 리스트를 만들 수 있음
  • 7. ol, ul -> ol = Ordered List / ul = Unordered List li의 부모 태그
  • 8. img = 이미지를 넣을 수 있다. 속성이란 것을 이용하면 더 정확히 표현가
  • 9. body = 본문, head = 제목, html = 모든 것의 부모, 독타입 html = 관용적으로 html위에 써준다.
  • 10. a href = "black" = 링크를 달 수 있다.
